# Security Policy

## Supported Versions

dflasher is pre-release software. Security fixes are handled on the default
branch until the project publishes stable releases.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory
for this repository. If advisories are unavailable, open a minimal issue that
does not include exploit details and ask for a private contact path.

Please do not publish working exploits or private model credentials in issues,
pull requests, or logs.
